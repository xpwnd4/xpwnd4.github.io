# Martin GitHub Pages Starter

A Jekyll-based GitHub Pages starter for cybersecurity blogging, writeups, projects, and resources.

## Included sections
- Blog posts in `_posts/`
- Writeups in `_writeups/`
- Projects in `_projects/`
- Resources in `_resources/`

## How to publish
1. Create a public GitHub repository named `yourusername.github.io`
2. Upload all files from this starter
3. Go to **Settings > Pages**
4. Set the source to **Deploy from a branch**
5. Choose the `main` branch and `/ (root)`
6. Save and wait for the site to publish

## Before publishing
- Replace `yourusername` placeholders
- Update About page and contact links
- Replace sample content with your own posts and writeups
- Add screenshots to `assets/images/`
